# OLED-1306-I2C-with-VSQUADRON-FM-
FPGA-based SSD1306 OLED (0.96", 128×64) I²C interface implemented on the VSQUADRON FM platform using Verilog HDL. The design includes a synthesizable I²C master, proper SSD1306 initialization, command/data mode control, monochrome display operation, and periodic data refresh for reliable graphic display interfacing and digital design practice.
